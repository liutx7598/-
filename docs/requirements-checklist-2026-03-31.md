## 1. 筛选技术指标与界面

- ◐ 根据指标类型做下拉菜单、自定义输入、范围输入等选项
  - 当前情况：前端已经支持按条件类型选择，并支持下拉菜单、数值输入、自定义参数输入。
  - 已有能力：均线条件、MACD、BOLL、KDJ、RSI、BIAS、Volume、形态匹配都可以在界面中添加。
  - 已补充：MA 斜率、MACD、BOLL 带宽、KDJ、RSI、BIAS、成交量倍数已经支持区间范围模式。
  - 仍有差距：并不是所有条件都做成了完整的“区间范围输入”控件，像 MACD/KDJ 交叉、BOLL 位置这类枚举型条件仍以下拉选择为主。
- ☑ 有重置条件选择和已选条件显示
  - 当前情况：界面已支持已选条件统一展示、单项移除、重置条件。
- ☑ 增加“我的选币策略”选项，可收藏常用筛选条件记录
  - 当前情况：已支持策略保存、收藏、策略列表、策略说明、策略执行间隔设置。
- ☑ 增加自动/手动筛选，可选择选币策略，可定时间
  - 当前情况：已支持手动扫描、自动监控、策略自动执行开关、策略执行间隔，以及后端策略运行状态展示。

## 2. 条件选股

### 2.1 K 线周期
- ☑ 支持 分时 / 1m / 5m / 10m / 15m / 30m / 1h / 2h / 3h / 4h / 8h / 12h / 1d / 2d / 3d
  - 当前情况：已支持独立的 `分时` 周期键，并与 `1m` 分开展示；底层行情仍复用 Gate 的 1m 数据源。

### 2.2 均线级别
- ☑ 支持 分时均线（分时专用）、MA5、MA10、MA20、MA30、MA60、MA120
  - 当前情况：后端已计算分时均线（同交易日逐根累计均价），前端条件编辑器已支持在分时均线与 MA5/10/20/30/60/120 之间切换。

### 2.3 技术指标
- ☑ 支持 MACD / BOLL / KDJ / RSI / BIAS / Volume
  - 当前情况：后端已实现指标计算，前端已支持将这些指标加入筛选条件。

## 3. 涨跌幅提醒
- ☑ 支持 1 分钟 / 5 分钟 / 1 小时 / 4 小时 / 今日 的涨跌幅提醒
  - 当前情况：后端已持续计算对应窗口涨跌幅，结果表已展示今日 / 1h / 5m 涨幅信息。
- ☑ 支持大于 / 小于阈值提醒
  - 当前情况：支持 gt / gte / lt / lte 四种阈值判断。
- ☑ 支持弹窗与声音预警
  - 当前情况：前端已支持浏览器通知和声音提醒，后端已支持持续运行的价格提醒判断。

## 4. 均线指标筛选
- ☑ 均线收拢
- ☑ 均线向上 / 向下
- ☑ 均线斜率（正负）
- ☑ 均线上穿 K 线实体
- ☑ 均线下穿 K 线实体
- ☑ 均线上穿 / 下穿均线（小级别穿大级别）
- ☑ 均线发散度（MAX(均线组) - MIN(均线组)）
- ☑ 多均线粘合度
- ☑ 均线相悖（5 日均线跟选择的均线方向不一致）
- ☑ K 线在均线之上 / 之下
  - 当前情况：后端条件引擎已支持以上规则，且均线选择器已接入分时均线与各 MA 级别。

## 5. 币种流通市值 / 市值排名
