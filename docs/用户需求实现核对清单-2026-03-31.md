## 1. 筛选技术指标与界面

- `◐` 根据指标类型做下拉菜单、自定义输入、范围输入
  - 当前情况：前端已经支持按条件类型选择，并支持下拉菜单、数值输入、自定义参数输入。
  - 已有能力：均线条件、MACD、BOLL、KDJ、RSI、BIAS、Volume、形态匹配均可在界面中添加。
  - 未完全满足：还没有为所有指标都做成完整的“范围输入/区间输入”控件，部分指标仍是单值阈值输入。

- `☑` 有重置条件选择和已选条件显示
  - 当前情况：界面已支持已选条件统一展示、单项删除、重置条件。
  - 已有能力：可在主页面直接移除条件标签，支持重新载入当前配置。

- `☑` 增加“我的选币策略”选项，可收藏常用筛选条件记录
  - 当前情况：已支持策略保存、收藏、策略列表、策略说明、策略间隔设置。

- `☑` 增加自动/手动筛选，可选择选币策略，可定时间
  - 当前情况：已支持手动扫描、自动监控、策略自动执行开关、策略执行间隔。
  - 说明：后端已存在策略运行状态与调度记录。

## 2. 条件选股

### 2.1 K线周期

- `◐` 支持 `分时 / 1m / 5m / 10m / 15m / 30m / 1h / 2h / 3h / 4h / 8h / 12h / 1d / 2d / 3d`
  - 当前情况：已支持 `1m / 5m / 10m / 15m / 30m / 1H / 2H / 3H / 4H / 8H / 12H / 1D / 2D / 3D`
  - 未完全满足：`分时` 目前是以 `1m` 入口兼容展示，没有独立的“分时专用”逻辑；时间周期文案里还有个别编码问题待清理。

### 2.2 均线级别

- `◐` 支持 `分时均线（分时专用）、5、10、20、30、60、120`
  - 当前情况：后端指标计算已支持 `MA5 / MA10 / MA20 / MA30 / MA60 / MA120`
  - 未完全满足：`分时均线（分时专用）` 还没有独立建模。

### 2.3 技术指标

- `☑` 支持 `MACD / BOLL / KDJ / RSI / BIAS / Volume`
  - 当前情况：后端已实现指标计算，前端已支持将这些指标加入筛选条件。
