# 给 Codex 的可执行操作文件

请基于当前目录继续开发一个“OKX 永续合约多周期形态筛选助手”，按单页面前后端分离思路实现，目标是：

- 接入 OKX v5 公共市场数据接口
- 自动获取 `SWAP` 永续合约
- 对 `15m / 1H / 2H / 3H / 4H` 五个周期做筛选
- 规则为 `MA5 与 MA20 收拢 + MA5 抬头向上 + K 线上穿 MA5`
- 所有命中结果统一展示在一张表中
- 每行可查看叠加 `MA5 / MA20` 的 K 线图
- 支持手动刷新
- 支持每 15 分钟自动监控
- 支持微信 / 钉钉 / 通用 Webhook 去重提醒
- 不要实现自动交易或任何私有下单接口

严格按以下顺序完善：

1. 补齐项目骨架和文档
2. 完善 OKX 公共数据客户端
3. 完善指标模块
4. 完善扫描服务
5. 完善提醒系统
6. 完善后端 API
7. 完善前端主页面
8. 联调与测试
9. 补 README / `.env.example` / 部署文档

当前版本额外说明：

- `3H` 周期允许由 `1H` 本地聚合
- 当前版本已经实现规则筛选，但尚未接入任何外部 LLM 分析
- 若后续提供 `Qwen` 可用 API，再补“命中后二次分析 / 摘要”能力
